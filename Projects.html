A shell based script that was written to be more productive and efficient with Git Hub.

The script, when executed, will automatically configure, build, and launch the sys161 kernel utility for OS161. This Script is to be used on Linux based machines.

A simple graphical application that takes an a, b, and c input and applies the quadratic formula to those values and graphical returns the results.
